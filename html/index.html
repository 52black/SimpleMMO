class authserver.RegistrationHandler(application, request, **kwargs)[source]¶

RegistrationHandler creates Users.

POST /register¶

Creates a User in the AuthenticationServer’s database.

Example request:

POST /register HTTP/1.1

username=asdf&password=asdf

Example response:

HTTP/1.1 200 OK

Registration succeeded.
Parameters:
  • username – The name of the user to create.
  • password – The password to use for authenticating the user.
  • email – Optional. An email to associate with the user.
Status Codes:
  • 200 – Registration succeeded.
  • 400 – A required parameter was missing.
  • 401 – The User already exists.
